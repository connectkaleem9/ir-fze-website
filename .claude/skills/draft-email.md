# Skill: /draft-email

## Description
Drafts professional IR-FZE email responses for client enquiries, follow-ups, proposals, introductions, and thank-you messages.

## Usage
```
/draft-email <type> [context]
```

## Examples
```
/draft-email enquiry "client asked about feasibility study for Dubai hotel project"
/draft-email follow-up "sent proposal 5 days ago, no reply"
/draft-email proposal "covering our PMS proposal for Al Noor Group"
/draft-email introduction "reaching out to Aldar Properties about AI services"
/draft-email thank-you "after meeting with GEMS Education procurement team"
```

## Email Types
- `enquiry` — respond to an inbound service enquiry
- `follow-up` — chase a lead that has gone quiet
- `proposal` — cover letter accompanying a proposal document
- `introduction` — cold outreach to a new prospect
- `thank-you` — post-meeting or post-call message

## What This Skill Does
1. Takes the type and context
2. Drafts a professional email in IR-FZE brand voice
3. Includes subject line, full body, and signature block
4. Offers 2 alternative subject lines
5. Optionally drafts a follow-up version for 3 days later if no reply
